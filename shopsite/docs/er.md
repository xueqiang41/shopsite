# 数据实例
## 1.分类（Category）

| 字段        | 类型           | 长度 |说明 |
| ------------- |:-------------:| -----:|:--------:|
| 名称      | 字符 | 50 | 分类名称|
| Slug      | Slug      |  50 | 根据name生成的，用于生成页面URL，必须唯一|
| 描述 | Text      |     |分类描述|
| 是否激活      | 布尔 |  | 分类是否可用|
| Meta关键词      | 字符      |  255 | meta关键词，有利于SEO， 用逗号分隔|
| Meta描述 | 字符      |   255  |meta描述 |
| 创建时间      | datetime      |   | 创建的时间|
| 创建时间 | datetime      |     |更新的时间|

## 2.产品（Product）

| 字段        | 类型           | 长度 |说明 |
| ------------- |:-------------:| -----:|:--------:|
| 名称      | 字符 | 255 | 品牌名称|
| Slug      | Slug      |  255 | 根据name生成的，用于生成页面URL，必须唯一|
| 品牌      | 字符 | 50 | 产品品牌|
| 计量单位      | 字符 | 50 | 产品计量单位|
| 价格      | 浮点数 | （9，2） | 产品价格 |
| 老价格      | 浮点数 | （9，2） | 产品老价格|
| 图片      | Image |  | 产品图片|
| 是否激活      | 布尔 |  | 产品是否可用|
| 是否畅销      | 布尔 |  | 产品是否畅销|
| 是否推荐      | 布尔 |  | 产品是否推荐|
| 数量      | 整形 |  | 产品数量|
| 描述 | Text      |     |产品描述|
| Meta关键词      | 字符      |  255 | meta关键词，有利于SEO， 用逗号分隔|
| Meta描述 | 字符      |   255  |meta描述 |
| 创建时间      | datetime      |   | 创建的时间|
| 创建时间 | datetime      |     |更新的时间|
